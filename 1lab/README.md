## 1. Основні типи даних

У файлі створено прості змінні різних типів:

- **str** — рядки  
- **int, float** — числові значення  
- **list** — список з елементами різних типів  
- **dict** — словник з ключами та значеннями  
- **tuple** — кортеж  
- **set** — множина

Також продемонстровано роботу з константами `True`, `False`, `None` та виведення списку ключових слів Python.

## 2. Використання вбудованих функцій

Показано приклади роботи стандартних функцій:

- `abs()` — модуль числа  
- `round()` — округлення  
- інші приклади форматування рядків та порівнянь

## 3. Цикли

### 3.1 Цикл for
Перебір списку за індексами.

### 3.2 Цикл while  
Виконання коду до виконання умови.

### 3.3 Використання else після циклу  
Пояснюється логіка роботи конструкції.

## 4. Розгалуження (if / else)

Показано, як працюють умовні конструкції:

- стандартний `if / else`  
- тернарний оператор  
- генерація значень через `randint()` для прикладів

## 5. Обробка помилок (try / except / finally)

Є приклад обробки помилок при діленні на нуль:

- `try` — блок коду, який може викликати помилку  
- `except` — перехоплення та обробка помилки  
- `finally` — код, що виконується завжди

## 6. Контекст-менеджер with

Продемонстровано читання файлу через конструкцію:

```python
with open("README.md", "r") as f: 
```
Це показує:

- безпечне відкриття/закриття файлів
- автоматичне керування ресурсами
- обробку помилки, якщо файл не існує

- звичайними функціями (`def`)
