# Опис класу [MyName](Working_with_Classes_and_Objects.py)

Клас **MyName** демонструє основи об’єктно-орієнтованого програмування (ООП) у Python.  
Він містить класові змінні, атрибути екземпляра, методи різних типів, перевірку даних та роботу з файлами.

---

## Основні складові класу

### 1. Класова змінна

- `total_names`  
  Лічильник усіх створених об’єктів класу.  
  Збільшується при створенні кожного нового екземпляра.

### 2. Конструктор `__init__`

- Ініціалізує екземпляр класу.  
- Приймає ім’я (`name`) та домен для email (`domain`).  
- Якщо `name=None`, автоматично створюється об’єкт з ім’ям `"Anonymous"` через метод `anonymous_user`.  
- Перша буква імені завжди великою (`capitalize()`).  
- Підвищує `total_names` та присвоює унікальний `my_id`.

### 3. Властивості (property)

- `whoami` — повертає рядок з ім’ям користувача.  
- `my_email` — повертає email у форматі `<name>@<domain>`.  
- `full_name` — повертає інформацію в форматі `"User #<id>: <name> (<email>)"`.

### 4. Методи екземпляра

- `create_email(domain=None)` — створює email, можна змінити домен.  
- `count_letters()` — повертає кількість букв в імені.  
- `save_to_file(filename="users.txt")` — зберігає інформацію про користувача у [файл](/2lab/users.txt).

### 5. Методи класу

- `anonymous_user()` — створює об’єкт з ім’ям `"Anonymous"`.

### 6. Статичні методи

- `say_hello(message="Hello to everyone!")` — виводить привітання. Можна змінювати текст.

### 7. Валідація імені

- `validate_name(name)` — перевіряє, що ім’я містить лише літери, інакше піднімає `ValueError`.

---

## Демонстрація

- Створюємо об’єкти для імен з кортежу `names`.  
- Використовуємо методи та властивості для отримання імені, email, кількості букв, повної інформації.  
- Викликаємо статичний метод `say_hello` з кастомним текстом.  
- Записуємо дані у файл `users.txt`.  
- Порівнюємо кількість імен у списку та загальну кількість об’єктів (через `total_names`) — різниця через створення Anonymous для `None`.

---

## Основна ідея

Цей клас показує, як:

- поєднувати класи, властивості та методи;  
- контролювати та перевіряти дані;  
- працювати зі статичними і класовими методами;  
- зберігати дані у файли;  
- будувати універсальні та безпечні об’єкти.
